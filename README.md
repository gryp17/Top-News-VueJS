# Top-News-VueJS

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run build-dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Back end Project setup
```
cd $MYPROJECT_DIR/api
npm install
```

### Starts the backend/API server
```
npm run start-server
```
